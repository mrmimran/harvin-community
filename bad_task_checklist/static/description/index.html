Module Task Checklist allows user to create checklists, add checklist items, and associate these checklists with individual tasks. This module provides a structured way to manage and track task-related activities, making task management more organized and efficient.

The key functionalities of the Task Checklist module:

Checklist Creation
Checklist Item Creation
Checklist Item Stages
Progress Tracking
Task Association
Improved Task Management

We can access Checklist under menu item Project/Configuration

Checklist created with checklist items

Checklist Item tree view

Add Checklist in Task, User can also add more than one Checklist in single task.

User can also use same Checklist in Multiple tasks
